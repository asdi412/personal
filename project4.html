Project 4: Social Media App

This project was about developing a social media platform that allows users to create profiles, post updates, follow other users, and engage through likes and comments. The application also features real-time notifications and a private messaging system.

Technologies Used

  • Frontend: React.js, CSS3
  • Backend: Node.js, Express.js, MongoDB
  • Real-time notifications: Socket.io

Features

  • User authentication with JWT (JSON Web Tokens)
  • Post creation, editing, and deletion
  • Follow/unfollow functionality
  • Real-time notifications for likes, comments, and follows
  • Private messaging with read receipts
  • Profile customization with photo upload

Challenges and Solutions

A significant challenge in this project was implementing real-time features like notifications and messaging. To solve this, I used Socket.io to establish a WebSocket connection between the client and server, ensuring instantaneous updates and messages.

Conclusion

This project was a deep dive into real-time web applications and social networking features. It greatly enhanced my understanding of user interaction design and scalable backend development.
